Certificate 3 & 4 in IT

We don't sit and memorize textbooks. We learn by doing, borrow ideas from GitHub and the wider internet, and the school funds the IT projects we want to take on.

In the Certificate 3 and 4 IT program at MVHS we work with real hardware, configure real networks, build websites from scratch, manage Linux servers and design databases. Our class even set up its own Minecraft server that runs 24/7.


Each of us is strongest in a different area, so we teamed up and built this site together to show what we picked up along the way.
